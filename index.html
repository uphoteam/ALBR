UPHO — Ekskul Komputer untuk Semua Siswa

UPHO adalah ektrakurikuler yang memfokuskan siswa untuk memahami dunia komputer secara praktis: dari pengenalan hardware & software, proses editing multimedia, hingga dasar-dasar coding. Semua kegiatan dilakukan secara kolaboratif agar setiap siswa punya pengalaman nyata.

Kegiatan & Pekerjaan Siswa

Pengenalan Komputer

  • Mengenal bagian PC: CPU, RAM, Storage, GPU, motherboard.
  • Installasi sistem operasi dasar dan pengaturan awal.
  • Pemeliharaan: pembersihan, instal antivirus, backup.
  • Dasar jaringan lokal (LAN) dan sharing file.

Proses Editing

  • Pengambilan footage & bahan (basic shooting).
  • Editing video: potong, transisi, teks, color grading sederhana.
  • Editing audio: noise removal, mixing, export untuk podcast.
  • Desain grafis: poster, thumbnail, materi presentasi.

Coding & Kolaborasi

  • HTML & CSS untuk membuat halaman profile atau dokumentasi.
  • JavaScript dasar: interaksi, form validation, dan DOM.
  • Proyek kelompok: website dokumentasi ekskul, aplikasi tugas sederhana.
  • Penggunaan Git (konsep) dan menyimpan project di repositori bersama.

Tanggung Jawab Rutin

  • Rotasi pemeliharaan lab (bersih, cek kabel, update software).
  • Backup dan upload project ke cloud atau server sekolah.
  • Dokumentasi hasil kerja: tiap proyek wajib punya README dan demo.
  • Mentoring antar siswa: yang mahir membantu yang pemula.
